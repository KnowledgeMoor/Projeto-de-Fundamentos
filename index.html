Sistemas de numeração : O que são?

Um numeral é um símbolo ou grupo de símbolos que representa um número em um determinado instante da evolução do homem. Tem-se que, numa determinada escrita ou época, os numerais diferenciaram-se dos números do mesmo modo que as palavras se diferenciaram das coisas a que se referem. Os símbolos "11", "onze" e "XI" (onze em romano) são numerais diferentes, representativos do mesmo número, apenas escrito em idiomas e épocas diferentes.

Um sistema de numeração, (ou sistema numeral) é um sistema em que um conjunto de números são representados por numerais de uma forma consistente. Pode ser visto como o contexto que permite ao numeral "II" ser interpretado como o numeral romano para dois, o numeral binário para três ou o numeral decimal para onze.

Binário

O sistema binário ou de base 2 é um sistema de numeração posicional em que todas as quantidades se representam com base em dois números, ou seja, zero e um. Os computadores digitais trabalham internamente com dois níveis de tensão, pelo que o seu sistema de numeração natural é o sistema binário.

Decimal

O sistema decimal é um sistema de numeração de posição que utiliza a base dez. Um sistema de numeração é um conjunto de princípios constituindo o artifício lógico de classificação em grupos e subgrupos das unidades que formam os números.

Hexadecimal

Hexadecimal é o sistema numérico de base 16, denotado utilizando os símbolos 0–9 e A–F (ou a– f ). Por exemplo, o número decimal 79 (cuja representação binária é 01001111) pode ser escrito como 4F em hexadecimal (4 = 0100, F = 1111). Cada dígito hexadecimal consegue representar 4 bits.

Exemplos de conversões

Decimal para binário :

  • Considere o 53 a ser convertido;
  • 53/2 = 26 e resto 1 - anote o resto: 1
  • 26/2 = 13 e resto 0 - anote o resto: 0
  • 13/2 = 6 e resto 1 - anote o resto: 1
  • 6/2 = 3 e resto 0 - anote o resto: 0
  • 3/2 = 1 e resto 1 - anote o resto: 1
  • 1/2 = 0 e resto 1 - anote o resto: 1

Binário para decimal :

  • Por exemplo, o número binário 1011;
  • (1 x 23) + (0 x 22) + (1 x 21) + (1 x 20);

Decimal para hexadecimal :

  • Considere o número decimal 242;
  • 242/16 = 15 e resto 2 - anote o resto: 2
  • 15/16 = 0 e resto 15 - anote o resto: 15
  • 0/16 = 0 e resto 0 - anote o resto: 0
  • Organizando do ultimo ao primeiro temos o números decimais: 0, 15, 2
  • Então temos: F2;

Hexadecimal para decimal :

  • Considere o número hexadecimal F2;
  • Identifique cada digito hexadecimal e seu valor decimal correspondente:
  • F = 15
  • 2 = 2
  • (15 x 161) + (2 x 160) = 240 + 2 = 242
